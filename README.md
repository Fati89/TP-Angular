## Résumé

| Composant | Rôle |
|-----------|------|
| Home | Page d'accueil |
| Product | Liste et suppression des produits |
| ProductService | Gestion des données (local / HTTP) |

## Backend utilisé (optionnel)

- GET : `http://localhost:8083/products`
- DELETE : `http://localhost:8083/products/{id}`
